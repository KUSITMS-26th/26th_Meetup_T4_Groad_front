# 🌳 Groad, 그로드
> 대학생들의 효율적이고 효과적인 **'진로 설계'** 에 **'도움'** 을 주는 서비스입니다.
> 한 그루의 나무가 자라 열매를 맺고, 여러 그루의 나무가 숲을 이룰 때까지 Groad는 대학생들을 돕겠습니다.
